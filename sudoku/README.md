As regras do Sudoku são simples: o objetivo é preencher uma grade de 9x9 com os números de 1 a 9, sem que haja repetições em cada linha, coluna ou bloco 3x3. Cada número deve aparecer apenas uma vez em cada linha, coluna e região. 
Como jogar:
1. Início:
A tabela do Sudoku já começa com alguns números preenchidos, que servem como pistas. 
2. Preenchimento:
Preencha as células vazias com os números de 1 a 9, respeitando as seguintes regras:
Linhas: Cada linha deve conter todos os números de 1 a 9, sem repetições. 
Colunas: Cada coluna deve conter todos os números de 1 a 9, sem repetições. 
Blocos: Cada bloco 3x3 deve conter todos os números de 1 a 9, sem repetições. 
3. Solução:
A solução é única, e cada número aparece apenas uma vez em qualquer linha, coluna ou bloco. 
Dicas:
Comece pelo fácil:
Se você é novo no Sudoku, comece por um nível de dificuldade fácil. 
Analise as linhas e colunas:
Identifique as linhas e colunas com poucos números preenchidos, pois elas podem ser mais fáceis de completar. 
Use notas:
Você pode anotar as possíveis opções para cada célula vazia, para facilitar a resolução. 
Procure por números repetidos:
Se você notar que um número já está presente em uma linha ou coluna, você pode excluí-lo como opção para as células vazias nessa linha ou coluna. 
Pense logicamente:
O Sudoku é um jogo de lógica, então tente pensar nas possíveis combinações de números para encontrar a solução.
